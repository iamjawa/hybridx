# HybridX Beta Testing Guide

Thank you for participating in the HybridX beta!

## What is HybridX?

HybridX is a breeding management platform for serious plant breeders. Track pedigrees, record crosses, evaluate seedlings, and manage your entire breeding programme from a single workspace.

## What's Included

- Plant management
- Cross planning and tracking
- Seed batch lifecycle (harvest → stratification → germination)
- Seedling evaluation with keyboard-driven quick mode
- Breeding goals with weighted trait matching
- Pedigree explorer
- Cross planner
- Dashboard with analytics
- QR code label generation
- CSV import/export

## How to Start

1. Visit the landing page and click "Explore Demo" to see a sample rose breeding programme
2. Click "Get Started" to create your account
3. Follow the onboarding wizard to set up your first species, plant, cross, and goal
4. If you have existing plants in a spreadsheet, use the Import page

## Known Limitations

- Image upload is URL-based (paste an image URL from any hosting service)
- Mobile experience is best on scan pages and evaluation mode
- Single-user only (team features coming post-beta)
- No offline mode

## Test Scenarios

### New Breeder
1. Complete onboarding wizard
2. Create 3 plants
3. Create a cross between two plants
4. Create a seed batch
5. Evaluate a seedling using Quick Evaluation
6. Create a breeding goal
7. Run goal matching

### Existing Breeder
1. Download a CSV template from the Import page
2. Populate with your plants
3. Upload and map columns
4. Review validation results
5. Import
6. Verify your plants appear

### Physical Garden
1. View a plant and click "Label"
2. Print the QR label
3. Scan with your phone
4. View the plant details on mobile
